# 🚗 ККИ: Полная коллекция ВАЗ — Калина

**Коллекционная карточная игра (ККИ)**, посвящённая легендарным автомобилям Волжского автозавода. Собирайте **21 модель** из четырёх исторических веток, улучшайте карты через крафт, участвуйте в битвах и открывайте уникальные достижения.

## 🎮 Игровой процесс

Вы — коллекционер классических автомобилей ВАЗ. Ваша цель — собрать **все 21 модель** в трёх вариантах:
- **Обычные карты** — базовая редкость.
- **Золотые карты** — улучшенная версия (10 обычных → 1 золотая).
- **Алмазные карты** — элитная версия (5 золотых → 1 алмазная).

- **Продажа обычной** — $50

- Каждый клик = +$5

## 🎛️ Режимы игры

### 1. 📚 **ВСЕ КАРТЫ**
- Просмотр полного списка всех 21 автомобиля с описанием, характеристиками и ценами
- Сохранение **недоступно**, коллекция не изменяется

### 2. 🎴 **КОЛЛЕКЦИЯ**
- Основной режим: покупка, продажа, крафт, битвы
- Деньги ограничены, требуется зарабатывать
- Прогресс **автоматически сохраняется**

### 3. ♾️ **БЕСКОНЕЧНЫЕ ДЕНЬГИ**
- Деньги не тратятся
- Подходит для исследования крафта и битв без ограничений

### 4. 🧪 **ТЕСТОВЫЙ РЕЖИМ**
- Бесконечные деньги + специальная панель
- Можно **выдать любую карту** (обычную, золотую, алмазную) без затрат
- Крафт **не требует материалов**
- Полезно для проверки всех механик и веток

## 💾 Слоты сохранения (5 штук)

- Каждый слот хранит: деньги, коллекцию, достижения и режим
- **Загрузка слота** — клик по слоту в верхней панели после старта игры
- **Сохранение** — кнопка «💾 Сохранить» под слотом

> Внимание: в режиме «Все карты» сохранение не работает — это чистая галерея.

## 🛠️ Крафт (улучшение карт)

- **10 обычных → 1 золотая** (требуется 10 одинаковых обычных карт)
- **5 золотых → 1 алмазная** (требуется 5 одинаковых золотых карт)

В **тестовом режиме** крафт доступен без материалов.

Выпавшая карта всегда **обычная** (золотые и алмазные получаются только крафтом).

## 🔧 Возможные проблемы и их решение

| Проблема | Решение |
|----------|---------|
| Картинки машин не загружаются | Игра автоматически подставляет иконку 🚗, функционал не страдает |
| Не сохраняется прогресс | Убедитесь, что вы не в режиме «Все карты» |
| Не открывается крафт | Карта должна быть в коллекции хотя бы в одном экземпляре |
| Тестовое меню не появляется | Запустите слот в режиме «Тестовый режим» (кнопка 🧪) |
| Не показываются веточные достижения | Соберите все модели соответствующей ветки |

## 📝 Примечания для разработчиков

- Игра полностью на чистом HTML/CSS/JS, без внешних зависимостей
- Для добавления новых моделей нужно расширить:
  - `CARDS_DATA` — игровые параметры
  - `FULL_CARD_INFO` — описательная информация
  - `BRANCHES` — распределение по веткам
- Вероятности выпадения регулируются в функции `buyRandomCard()`
- Достижения проверяются автоматически через `checkAchievements()`
- Ключ сохранения: `car_collection_game_slots_v13`
