# 🚗 ККИ: Полная коллекция ВАЗ — все ветки

**Коллекционная карточная игра (ККИ)**, посвящённая легендарным автомобилям Волжского автозавода. Собирайте **24 модели** из **пяти исторических веток**, улучшайте карты через крафт, участвуйте в битвах и открывайте уникальные достижения. Каждая ветка представлена в виде **визуального дерева**!

## 🎮 Игровой процесс

Вы — коллекционер автомобилей ВАЗ. Ваша цель — собрать **все 24 модели** в трёх вариантах:
- **Обычные карты** — базовая редкость.
- **Золотые карты** — улучшенная версия (10 обычных → 1 золотая).
- **Алмазные карты** — элитная версия (5 золотых → 1 алмазная).

## 🎛️ Режимы игры

### 1. 📚 **ВСЕ КАРТЫ**
- Просмотр полного списка всех 24 автомобилей с описанием, характеристиками и ценами
- Сохранение **недоступно**, коллекция не изменяется

### 2. 🎴 **КОЛЛЕКЦИЯ**
- Основной режим: покупка, продажа, крафт, битвы
- Деньги ограничены, требуется зарабатывать
- Прогресс **автоматически сохраняется**

### 3. ♾️ **БЕСКОНЕЧНЫЕ ДЕНЬГИ**
- Деньги не тратятся
- Подходит для исследования крафта и битв без ограничений

### 4. 🧪 **ТЕСТОВЫЙ РЕЖИМ**
- Бесконечные деньги + специальная панель
- Можно **выдать любую карту** (обычную, золотую, алмазную) без затрат
- Крафт **не требует материалов**
- Полезно для проверки всех механик и веток

## 🛠️ Крафт (улучшение карт)

- **10 обычных → 1 золотая** (требуется 10 одинаковых обычных карт)
- **5 золотых → 1 алмазная** (требуется 5 одинаковых золотых карт)

В **тестовом режиме** крафт доступен без материалов.

## 🌳 Визуальное дерево веток

При выборе ветки в разделе «ВЕТКИ» вы увидите **вертикальное дерево** с соединительными линиями между моделями. Порядок моделей соответствует их историческому появлению:

| Ветка | Порядок моделей |
|-------|-----------------|
| **Классические ВАЗ** | 2101 → 2102 → 2103 → 2104 → 2105 → 2106 → 2107 |
| **ВАЗ Самара** | 2108 → 2109 → 21099 → 2115 → 2114 → 2113 |
| **ВАЗ 110** | 2110 → 2111 → 2112 |
| **ВАЗ Калина** | 1118 → 1119 → 1117 → 2192 → 2194 |
| **ВАЗ Приора** | 2170 → 2172 → 2171 |

Каждый узел дерева — это карта автомобиля. Полученные карты подсвечиваются золотым или алмазным градиентом. Заблокированные карты отображаются с затемнением и иконкой 🔒.

## 🎨 Особенности интерфейса

- **Цветовая индикация редкости**:
  - Серая рамка — обычная карта
  - Голубая рамка — редкая карта
  - Оранжевая рамка — легендарная карта (только 2101)

- **Визуальное дерево**: вертикальная структура с соединительными линиями, показывающая развитие каждой ветки

- Адаптивный дизайн — игра работает на ПК и мобильных устройствах

## 🔧 Возможные проблемы и их решение

| Проблема | Решение |
|----------|---------|
| Картинки машин не загружаются | Игра автоматически подставляет иконку 🚗, функционал не страдает |
| Не сохраняется прогресс | Убедитесь, что вы не в режиме «Все карты» |
| Не открывается крафт | Карта должна быть в коллекции хотя бы в одном экземпляре |
| Тестовое меню не появляется | Запустите слот в режиме «Тестовый режим» (кнопка 🧪) |
| Ветка «ВАЗ 110» не отображается | Обновите страницу (баг исправлен в версии v15) |

## 📝 Примечания для разработчиков

- Игра полностью на чистом HTML/CSS/JS, без внешних зависимостей
- Для добавления новых моделей нужно расширить:
  - `CARDS_DATA` — игровые параметры
  - `FULL_CARD_INFO` — описательная информация
  - `BRANCHES` — распределение по веткам (с порядком моделей для дерева)
- Вероятности выпадения регулируются в функции `buyRandomCard()`
- Достижения проверяются автоматически через `checkAchievements()`
- Ключ сохранения: `car_collection_game_slots_v15`
- Визуализация дерева: функция `showBranchTree()`
